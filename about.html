Tentang Saya

Selamat datang! Saya adalah seorang pengembang web dengan passion dalam desain antarmuka pengguna dan pengalaman pengguna. Saya berfokus pada membuat aplikasi web yang indah, fungsional, dan intuitif.

Saya memiliki pengalaman dalam teknologi seperti HTML, CSS, JavaScript, dan framework seperti ReactJS. Saat ini, saya sedang mengeksplorasi AI dan machine learning untuk mengembangkan solusi inovatif.

Jika Anda tertarik untuk berkolaborasi atau memiliki proyek yang menarik, jangan ragu untuk menghubungi saya melalui formulir kontak di bawah atau melalui media sosial.
